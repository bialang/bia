Detailed Description

template<typename Default, typename... Types>
class bia::utility::virtual_object< Default, Types >

Holds one object of different type.

Holds up to one object of the specified types in Types. The object is stored in local memory.

Template Parameters
DefaultThe default return type.
TypesThe valid types.
